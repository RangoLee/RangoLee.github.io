Mac 安装Python3.5.2

发表于 2016-12-20   |   分类于 python3   |     |   阅读次数

1.安装Python3.5.2

  • MacOS系统10.12.2，终端运行brew install python3
  • 报警告
  • 然后开始下载，中间可能有些包需要科学上网才能下载下来。
  • 安装失败，科学上网后继续brew install python3
  • 这次又报错了,没有brew link
  • 再次尝试,警告⚠️Warning: python3-3.5.2 already installed, it’s just not linked
  • 使用brew doctor查看错误信息

control + c 退出

  • 根据提示信息做如下修改：chown,export PATH,brew link

  • chown

  • export PATH=
  • brew link
  • 至此安装成功！！！
