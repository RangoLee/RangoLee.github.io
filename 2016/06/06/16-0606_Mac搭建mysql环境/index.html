Mac OS 搭建 mysql 环境

发表于 2016-06-06   |   分类于 iOS   |     |   阅读次数

1. MySQL Wiki

  1. 双击 mysql-5.6.12-osx10.7-x86_64.dmg
  2. 双击 mysql-5.6.12-osx10.7-x86_64.pkg
  3. 双击 MySQL.prefPane
  4. 系统偏好设置里启动 MySQL

  1. 验证

  2. 登入MySQL服务器,首次登入无密码,直接回车即可 看到下面这些说明安装成功.

  3. 验证数据库,四个是正常的,如果没有四个重启电脑,开启服务应该就是4个了

  4. 退出

  5. 修改MySQL 的 root 密码 为123456 (次数有限,二次修改很麻烦)

  6. 输入新密码登入

2. 安装图形化界面管理软件

  1. 双击 mysql-workbench-gpl-5.2.47-osx-i686.dmg
  2. 启动,配置数据库

  3. 创建数据库（数据库的名字和后面要和服务器关联的名字一致就可以。应该是openfiredb，这里写错了，后面也将用openfiledb这个名字）

  4. 激活目标数据库，用于后面和服务器关联。

  5. 创建表(运行脚本)-刷新


3.安装 JDK

  1. 检查一下

  2. 安装 JDK（Java Development Kit）去官网下载安装即可。
