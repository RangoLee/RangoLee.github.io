Mac 安装 cmake

发表于 2016-05-23   |   分类于 Mac   |     |   阅读次数

1 安装包安装 cmake

  1. 从官网下载cmake安装包。默认安装
  2. 链接cmake.app 与 terminal
    1. 打开终端，输入export PATH=/Applications/CMake.app/Contents/bin:$PATH
  3. 配置完成。（以上链接只对本终端窗口有效）

2 使用 homebrew 安装（推荐）

  1. 安装homebrew

  2. 使用 brew 安装 cmake
