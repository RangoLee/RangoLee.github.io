Mac终端(Terminal)常用命令

2016-05-20
Mac
约 402 字 预计阅读 1 分钟

文章目录

1. 使用caffeinate阻止Mac运行屏幕保护和睡眠

caffeinate 能阻止 mac 进入睡眠状态和屏保的的激活。可以用-t命令为命令加入具体时间。例如输入如下命令可以使 mac 一小时不进入睡眠状态。

1

caffeinate -t 3600

2. 使用 pkgutil解压 PKG 文件

使用下面的命令可以将 music.pkg解压到桌面。

1

pkgutil --expand music.pkg ~/Desktop/

3. 使用 purge 命令释放内存

purge 命令可以清除内存和硬盘的缓存，与重启 Mac 的效果差不多。purge 命令可以让不活跃的系统内存转变为可以使用的内存，

1

purge

4. 使用 open 命令开启多个相同应用

open 命令可以在终端中打开应用，使用-n可以开启多个相同应用。

1

open -n /Application/Safari.app/

5. 不通过 App Store 更新 OS X

想要更新系统却不想打开 App Store ，下面的命令就可以使用终端升级 OS X

1

sudo softwareupdate -i -a

6.获得某个文件或文件夹的路径

直接将任意文件或文件夹拖入到终端窗口中，即可以看到文件的详细路径。

7.显示隐藏的文件和文件夹

1
2

defaults write com.apple.finder AppleShowAllFiles YES
killall Finder

1
2

defaults write com.apple.finder AppleShowAllFiles NO
killall Finder

8.Mac时间机器的本地备份的开启与关闭

1
2

sudo tmutil disablelocal
sudo tmutil enablelocal
