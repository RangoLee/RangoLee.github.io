5.C#变量

一个变量只不过是一个供程序操作的存储区的名字。在C#中，每个变量都有一个特定的类型，类型决定了变量的内存大小和布局。范围内的值可以存储在内存中，可以对变量进行一系列的操作 C#提供的基本数据类型大致如下：

类型 举例
整数类型 sbyte、byte、short、ushort、int、uint、long、ulong和char
浮点数 floa 和 double
十进制类型 decimal
布尔类型 true或false，指定的值
空类型 可为空值的数据类型
C#允许定义其他值类型的变量，比如enum也允许定义引用类型变量，比如class。

C#中的变量定义

C#中变量定义的语法

1

<data_type> <variable_list>;

在这里，data_type必须是一个有效的C#数据类型，可以是char、int、float、double或其他自定义类型的数据类型。variable_list可以由一个或多个逗号分隔符的标识符名称组成。

1
2
3
4
5

int i, j, k;
char ch;
float salary;
double d;
int x = 100; // 定义的时候初始化

C#中变量的初始化

变量通过在等号后跟一个常量表达式进行初始化（赋值）。variable_name = value; 变量可以在声明时被初始化（指定一个初始值）。初始化由一个等号后跟一个常量表达式组成。

1
2
3
4
5
6

data_type variable_name = value;
// 
int nRet = 0;
byte bt = 22;
double d = 3.14;
char ch = 'c';

正确的初始化变量是一个良好的编程习惯，以防止程序产生意想不到的结果。

1
2
3
4
5
6
7
8

short a;
int b;
double c;
// 初始化
a = 10;
b = 100;
c = a+b;
Console.WriteLine("a= {0}, b={1}, c={2}", a, b, c);

C#中的Lvalues 和 Rvalues

C#中的两种表达式：

  • lvalues：Lvalues表达式可以出现在赋值语句的左边或右边
  • rvalues：Rvalues表达式可以出现在赋值语句的右边，不能出现在赋值语句的左边。 变量是lvalues的，所以可以出现在赋值语句的左边。数值是rvalues的，因此不能被赋值，不能出现在赋值语句左边
1
2

int n = 20;// 有效语句
10 = n;// 无效的语句
