2.C#基本语法

C# 是一种面向对象的编程语言。在面向对象的程序设计方法中，程序由各种相互交互的对象组成。相同种类的对象通常具有相同的类型，或者说，是在相同的 class 中。 例如，以 Rectangle（矩形）对象为例。它具有 length 和 width 属性。根据设计，它可能需要接受这些属性值、计算面积和显示细节。 让我们来看看一个 Rectangle（矩形）类的实现，并借此讨论 C# 的基本语法：

using System;
namespace RectangleApplication
{
 class Rectangle
 {
  // 成员变量
  double dLength;
  double dWidth;
  public void Acceptdetails()
  {
   dLength = 4.5;
   dWidth = 3.5;
  }
  public double GetArea()
  {
   return dLength * dWidth;
  }
  public void Display()
  {
   Console.WriteLine("Length:{0}",dLength);
   Console.WriteLine("Width:{0}",dWidth);
   Console.WriteLine("Area:{0}",GetArea());
  }
 }
 class ExecuteRectangle
 {
  static void Main(string[] args)
  {
   Rectangle rect = new Rectangle();
   rect.Acceptdetails();
   rect.Display();
   Console.ReadKey(); 
  }
 }
}

成员变量

变量是类的属性或数据成员，用于存储数据。在上面的Rectangle类有两个成员变量，length和width。

成员函数

函数是一系列执行指定任务的语句。类的成员函数是在类内声明的。我们举例的类Rectangle包含了三个成员函数：AcceptDEtail、GetArea和Display。

实例化一个类

在上面的程序中，类ExecuteRectangle是一个包含Main()函数和实例化Rectangle类的类。

标识符

标识符是用来识别类、变量、函数或者任何其它用户定义的项目。在C#中，类的命名必须遵循如下基本原则：

1,标识符必须以字母或下划线_开头，后面可以跟一系列的字母、数字（0-9）或者下划线_。标识符中的第一个字符不能使数字。
标识符必须不包含任何嵌入的空格或者符号，比如：? - +! @ # % ^ & * ( ) [ ] { } . ; : " ' / \。但是，可以使用下划线_.
标识符不能是C#关键字
