1.下载安装

1. 先去Lua官网下载最新版 lua

Rango-MBP:~ rango$ curl -R -O http://www.lua.org/ftp/lua-5.3.3.tar.gz

2. 然后解压压缩包

Rango-MBP:~ rango$ tar zxf lua-5.3.3.tar.gz

3. cd 到解压后的文件夹

Rango-MBP:~ rango$ cd lua-5.3.3

4. 然后编译测试

Rango-MBP:lua-5.3.3 rango$ make macosx test
